Welcome!
To set up your web site, use the following startup guide:
  1. Select A Theme: Choose the "Template" link to the left to select one of our pre-installed templates. Or, enter your own HTML code for a customized design.
  2. Add Pages: Choose the "Pages" link to the left to select the pages you'd like to appear on your site.
  3. Add Store Categories: Choose the "Categories" link to the left to start entering some categories to your online store. Use the arrow icons to move the categories' positions as they will appear on your site. You can set up to three levels of categories (main category > subcategory > end category).
  4. Add Products: Once your categories have been entered, choose the "Products" link to the left to start adding products. Add or edit products singly to modify all details, or edit the main points and modify up to 10 items at once. You can also upload text files of products if desired.
  5. Change Settings: Choose the "Site Settings" link to the right to update main site variables. Choose the "Fonts & Colors" link to update color values (if you are not using your own CSS file). Choose the "Catalog Setup" link to modify all shopping catalog variables as needed.
  6. Add Additional Features: If you want to add additional functionality to your web site, such as a gift registry, affiliate program, guest book, link page, event calendar, wholesale catalog, articles, frequently asked question list, gift certificates, order tracking or coupons, select the appropriate link and use the help screens to set up the feature. All additional features are optional and can be turned off.
  7. Back Up Your Data: Periodically, especially after making many changes or adding several products, you should back up your database. Use the "Backup" link to create a database dump copy. Save the file on your hard drive, then delete the file from your web site so that it is not visible to others.

To visit this screen again at any time, click on the home button at the top of the page.

If you need further help on any page, just click the help icon at the top of the page to read more information about that screen.
